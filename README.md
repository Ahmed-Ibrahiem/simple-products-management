# Products Dashboard

A simple **Products Dashboard** built with **React Class Components** as a training project.  
The main goal of this project is to practice **React Class Components**, **Context API**, and **lifecycle methods**, while building a responsive product management app.

---

## Features

- Show products list
- Add new product
- Edit existing product
- Delete product
- Filter and search products
- Fetch and display dummy data
- Simple data tracking
- LocalStorage integration
- Pagination (multi pages)
- Responsive design
- Toast notifications

---

## Tech Stack

- **React** (Class Components)
- **Context API** for state management
- **LocalStorage** for persistence
- **React-Toastify** for notifications
- **Lifecycle Methods** with Class Components

---

## Data

The app uses **dummy/fake data** for products (no real API).
